Local

Enabled login with keys:-

cat ~/.ssh/id_rsa.pub | ssh username@VPS_IP_ADD 'cat >> .ssh/authorized_keys'

These few rows in .ssh/config:-

host vps hostname VPS_IP_ADD user username Compression yes Ciphers blowfish-cbc,arcfour ControlMaster auto ControlPath /tmp/%r@%h:%p Installed Vim and here is my latest .vimrc:-

https://github.com/motyar/dotfiles/blob/master/.vimrc
